# PRD

## 项目定义

`ifome` 是一个面向求职场景的信息流 Agent，负责把招聘链接、聊天截图和文本消息转成可追踪、可排序、可提醒的行动项。

## 核心目标

- 降低用户处理求职信息的认知负担。
- 避免遗漏 DDL、笔试、面试和宣讲等关键时间点。
- 把处理链路做成可解释的单 Agent 工作流，而不是黑盒聊天。
- 同时作为工程作品，支持面试中的架构讲解与调试演示。

## POC 范围

- 接入：文本、链接、图片。
- 处理：归一化、分类、字段抽取、画像匹配、优先级打分、提醒规划。
- 展示：卡片列表、记忆管理、聊天控制台。
- 治理：结构化日志、trace_id、实验功能隔离。

## 非目标

- 自动投递岗位。
- 自动监听或自动操作微信私聊。
- 一开始就引入复杂多 Agent。
